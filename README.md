# Event Registration Module

## Installation
1. Place the `event_registration` folder in `web/modules/custom/`.
2. Go to Admin > Extend and enable "Event Registration".

## URLs
- Admin Config: `/admin/config/event-setup`
- Registration: `/event/register`
- Listing: `/admin/event-registrations`

## Features
- Custom Database Tables using Schema API.
- AJAX dependent dropdowns for event selection.
- Dependency Injection for database and mail services.
